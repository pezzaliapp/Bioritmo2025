# Bioritmo2025

Bioritmo2025 è una Progressive Web App (PWA) che consente di calcolare i bioritmi personali basati sulla data di nascita. Progettata per essere installabile su dispositivi desktop (Windows, macOS) e mobili (iOS, Android), l’app offre una grafica interattiva e un’esperienza completamente offline.

Caratteristiche principali
	•	Calcolo dei bioritmi: Scopri i tuoi cicli fisico, emotivo e intellettuale.
	•	Grafici interattivi: Visualizza i tuoi cicli in un grafico chiaro e intuitivo.
	•	Compatibilità multipiattaforma: Funziona su browser moderni, desktop e dispositivi mobili.
	•	Funzionalità offline: Grazie a un Service Worker, l’app è accessibile anche senza connessione.
	•	Aggiornamenti automatici: Ogni modifica al repository viene sincronizzata automaticamente.

Come utilizzare l’app
	1.	Visita la PWA a questo link: https://pezzaliapp.github.io/Bioritmo2025.
	2.	Inserisci la tua data di nascita e una data di riferimento per calcolare i bioritmi.
	3.	Visualizza i risultati sul grafico interattivo.
	4.	Installa l’app cliccando su “Aggiungi alla schermata Home” (mobile) o “Installa” (desktop).
